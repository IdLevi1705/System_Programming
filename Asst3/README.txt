Running the files:

1. run gcc to compile both file (Makefile is not ready yet).
2. when running server add arbitrary port (4000-9999) in order to host the client.
3. when running client file add IP as second argument and the same port as given by the server for third argument.

e.g
./server 9898

./client 127.0.0.1 9898
and start typing here....


note:
you have to open two different windows in order to see the communcation betweeen client and server.
